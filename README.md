# POWDER RUSH — パウダーラッシュ 🏂

ブラウザで動く3Dスノーボードゲーム。無限に生成される雪山を滑り降り、キッカーでトリックを決めてハイスコアを狙う。

![Three.js](https://img.shields.io/badge/Three.js-r160-blue) ![No Build](https://img.shields.io/badge/build-none-green)

## 遊び方

`index.html` をブラウザで開くだけ。ビルド不要・ネット接続不要（Three.jsはローカル同梱）。

ローカルサーバーで起動する場合：

```sh
node _server.js
# → http://localhost:8642
```

## 操作

| キー | 地上 | 空中 |
|---|---|---|
| ← → / A D | ターン | スピン（FS/BS 180〜1080） |
| ↑ / W | タック加速 | フロントフリップ |
| ↓ / S | ブレーキ | バックフリップ |
| SPACE | 長押しで溜めて、離すとジャンプ | — |
| SHIFT（+方向キー） | — | グラブ5種 |
| R | リスタート | |

### グラブ（空中でSHIFT、押しながら方向キーで種類が変わる）

| 操作 | グラブ |
|---|---|
| SHIFT のみ | INDY |
| SHIFT + ↑ | NOSE |
| SHIFT + ↓ | TAIL |
| SHIFT + ← | MELON |
| SHIFT + → | STALEFISH |

- **テーブルトップ・キッカー**：踏切→平らなテーブル→着地斜面の本格的な形状。スピードを乗せて飛ぶと大ジャンプ
- **レールセクション**：ジャンプして金属レールに飛び乗ると 50-50 グラインド。長く乗るほど高得点、SPACE で飛び降りる
- 着地時にボードの向きがズレすぎていると **WIPEOUT!**（中程度なら SKETCHY で減速のみ）
- トリックを連続で決めるとコンボ倍率アップ（最大 x9）
- 木や岩にかすった程度では転ばず弾かれて減速。高速の正面衝突のみクラッシュ
- スコアは「距離 x2 + トリック」

## 技術メモ

- 単一HTML + Three.js（依存はそれだけ）
- 地形は解析的ハイトフィールド（バリューノイズ3層 + 谷型コース + テーブルトップ・キッカーを高さ関数に焼き込み）— 見た目と当たり判定が完全一致
- レールも解析的に配置し、空中から上面付近に降りると自動でグラインド判定
- チャンク方式で無限生成、樹木・岩はInstancedMeshで描画
- カービング物理：重力の斜面投影 + ボード方向のエッジグリップ
- 関節を球でつないだ高解像度ライダーモデル（膝・肘の2節構造、ヘルメット・ゴーグル・バックパック）
- 雪煙・降雪パーティクル、WebAudioによる風切り音・カービング音
